# ToDo-App

A simple and responsive To-Do List web application built using **HTML, CSS, and JavaScript**.  
The app allows users to add tasks, mark them as completed, delete tasks, and automatically saves all data using **Local Storage**.

---

##  Features
- Add new tasks.
- Mark tasks as completed.
- Delete tasks.
- Automatic saving using browser Local Storage.
- Simple and clean user interface.

- **HTML5** (Page structure)  
- **CSS3** (Design & Layout)  
- **JavaScript** (Functionality & Logic)

  ## How to Use
1. Open `index.html` in any browser.
2. Add a new task from the input field.
3. Click a task to mark it completed.
4. Click the delete icon to remove a task.
5. Tasks are saved automatically and stay after refreshing.
